# xk6-icmp

> [!NOTE]
> Work in progress

k6 extension for [ICMP](https://en.wikipedia.org/wiki/Internet_Control_Message_Protocol)

## Build
```sh
# Install xk6
go install go.k6.io/xk6/cmd/xk6@latest

# Build k6 with this extension
xk6 build --with github.com/sh-cho/xk6-icmp
```
